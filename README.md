# Extractor Multimedia YouTube (Windows) 🚀

Una herramienta interactiva en formato `.bat` diseñada para facilitar la descarga de audio y video de internet, eliminando la necesidad de escribir comandos complejos en la terminal.

## 🛠️ ¿Cómo funciona?
Este proyecto es un menú visual que actúa como intermediario para ejecutar **yt-dlp** y **FFmpeg** de forma más cómoda. No necesitas recordar argumentos ni configuraciones de calidad; el script se encarga de todo de manera automatizada.

## ✨ Características
- Descarga de música en MP3 con selección de calidad (hasta 320 kbps).
- Descarga de video en MP4 (desde 720p hasta la máxima calidad disponible).
- Incorporación automática de carátulas (`thumbnails`) y metadatos a los archivos.
- Instalador automático de dependencias integrado mediante `winget`.
- Los archvios se guardaran en la misma carpeta desde donde ejecutes el archivo.
